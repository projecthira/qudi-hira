Layout
======
The file "scanfile.txt" contains the scan points as follows:


	total_x_points, total_y_points
	x1, y1
	x2, y1
	.
	.
	.
	-1, -1
	
	
Values
======
The data is shifted by 1 from what Nanonis is showing:

eg. 

scanfile.txt
	X, Y
	0, 0
	1, 0
	0, 1
	1, 1

Nanonis
	X, Y
	1, 1
	2, 1
	1, 2
	2, 2
